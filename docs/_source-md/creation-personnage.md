# Creation de personnage

Sources images : 1-7.

Statut : premiere extraction faite, a relire avec le MD.

## Vue d'ensemble

La creation se resume en cinq grandes decisions :

1. Choisir une classe.
2. Determiner les origines : historique, espece et langues.
3. Determiner les six valeurs de caracteristique.
4. Choisir un alignement.
5. Finaliser la fiche : aptitudes, maitrises, PV, CA, attaques, sorts, nom et details narratifs.

Pour votre table actuelle, les classes suivies sont :

| Classe | Role rapide | Caracteristique principale utile |
|---|---|---|
| Clerc | Magie divine, soutien, soin, protection ou chatiment. | Sagesse |
| Druide | Magie naturelle, soin, formes animales, controle d'environnement. | Sagesse |
| Rodeur | Combat martial, survie, exploration, magie sauvage. | Dexterite et Sagesse |

## Etape 1 - Classe

Noter sur la fiche :

- classe ;
- niveau ;
- points d'experience ;
- bonus de maitrise ;
- formations aux armures ;
- maitrises d'armes, outils, competences et jets de sauvegarde donnees par la classe ;
- aptitudes de classe du niveau 1.

Au niveau 1 :

- PX de depart : 0 ;
- bonus de maitrise : +2.

## Etape 2 - Origines

Une origine combine trois choses :

- un historique ;
- une espece ;
- deux langues en plus du commun.

L'historique est important parce qu'il donne notamment :

- un don d'origine ;
- des maitrises ;
- des caracteristiques recommandees ou modifiees.

L'espece donne des traits comme la taille, la vitesse et des aptitudes specifiques.

### Langues

Un personnage connait au minimum :

- commun ;
- deux langues courantes, choisies ou tirees au hasard selon la table.

Langues courantes visibles dans les sources :

- draconique ;
- elfique ;
- geant ;
- gnome ;
- gobelin ;
- halfelin ;
- langue multiverselle des signes ;
- nain ;
- orc.

Langues rares visibles dans les sources :

- abyssal ;
- argot des voleurs ;
- celeste ;
- commun des profondeurs ;
- druidique ;
- infernal ;
- primordial ;
- profond ;
- sylvestre.

Note : les langues rares semblent plutot venir d'aptitudes, de classes ou de choix specifiques. A verifier avec le MD avant de les autoriser librement.

### Ajustements d'historique

Apres attribution des valeurs, appliquer les bonus donnes par l'historique :

- soit +2 dans une caracteristique et +1 dans une autre ;
- soit +1 dans trois caracteristiques.

Limite : une valeur ne peut pas depasser 20 avec ces augmentations.

## Etape 4 - Alignement

L'alignement sert surtout a cadrer les valeurs, les convictions et le comportement du personnage.

Il combine :

- un axe moral : bon, neutre ou mauvais ;
- un axe d'ordre : loyal, neutre ou chaotique.

Options classiques :

- loyal bon ;
- neutre bon ;
- chaotique bon ;
- loyal neutre ;
- neutre ;
- chaotique neutre ;
- loyal mauvais ;
- neutre mauvais ;
- chaotique mauvais.

Point de table : les sources partent du principe que les PJ ne sont pas mauvais par defaut. A valider avec le MD si quelqu'un veut jouer un personnage mauvais.

## Etape 5 - Finalisation

### Checklist de fiche

Noter ou calculer :

- aptitudes de classe niveau 1 ;
- jets de sauvegarde maitrises ;
- competences maitrisees ;
- perception passive ;
- points de vie ;
- des de vie ;
- initiative ;
- classe d'armure ;
- armes et attaques ;
- DD de sauvegarde des sorts ;
- bonus d'attaque de sort ;
- sorts mineurs, sorts prepares et emplacements de sort si la classe en utilise ;
- nom ;
- details personnels.

### Formules utiles

| Element | Calcul |
|---|---|
| Jet de sauvegarde maitrise | modificateur de caracteristique + bonus de maitrise |
| Competence maitrisee | modificateur de caracteristique + bonus de maitrise |
| Perception passive | 10 + modificateur de Sagesse (Perception) |
| Initiative | modificateur de Dexterite |
| CA sans armure ni bouclier | 10 + modificateur de Dexterite |
| Attaque au corps a corps maitrisee | modificateur de Force + bonus de maitrise |
| Attaque a distance maitrisee | modificateur de Dexterite + bonus de maitrise |
| DD de sauvegarde des sorts | 8 + modificateur d'incantation + bonus de maitrise |
| Attaque de sort | modificateur d'incantation + bonus de maitrise |

Note : certaines armes, armures, boucliers ou aptitudes de classe peuvent modifier ces calculs.

## Questions narratives utiles

Pour finir le personnage, repondre rapidement a quelques questions :

- Qui l'a forme ou eduque ?
- Qui compte le plus pour lui ?
- Quelle est sa pire crainte ?
- Que cherche-t-il en partant a l'aventure : savoir, fortune, gloire, justice, misericorde, pouvoir ou autre chose ?
- Son historique et son espece racontent-ils une tension, une dette, une vocation ou une rupture ?
