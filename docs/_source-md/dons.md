## Dons par usage

## Prerequis

### Initie a la magie

Categorie : don d'origines.

| Benefice | Detail |
|---|---|
| Deux sorts mineurs | Choisir deux sorts mineurs dans la liste de sorts de Clerc, Druide ou Magicien. |
| Caracteristique d'incantation | Intelligence, Sagesse ou Charisme, choisie a l'acquisition du don. |
| Sort du 1er niveau | Choisir un sort du 1er niveau dans la meme liste que les sorts mineurs ; il est toujours prepare. |
| Lancement gratuit | Le sort du 1er niveau peut etre lance une fois sans depenser d'emplacement ; recuperation apres repos long. |
| Emplacements | Le sort du 1er niveau peut aussi etre lance avec les emplacements disponibles. |
| Changement de sort | A chaque niveau gagne, possibilite de remplacer un sort choisi par ce don par un autre sort du meme niveau dans la meme liste. |
| Repetable | Peut etre choisi plusieurs fois, avec une liste de sorts differente a chaque fois. |
