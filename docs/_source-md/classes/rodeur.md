# Rodeur

Sources images : 15-17.

Statut : premiere extraction faite. Sous-classes a completer si besoin.

## Role de la classe

Le rodeur combine combat martial, exploration, pistage, survie et magie naturelle.

Pour une table debutante, le rodeur sert souvent a :

- infliger des degats fiables avec armes ;
- couvrir l'exploration et la survie ;
- utiliser une bonne Dexterite en combat et en discretion ;
- apporter une magie utile sans etre un lanceur de sorts principal ;
- traquer une cible prioritaire avec `marque du chasseur`.

## Niveau 1

Au niveau 1, le rodeur recoit :

- les traits de base ;
- l'aptitude Sorts ;
- l'aptitude Bottes d'arme ;
- l'aptitude Ennemi jure.

### Sorts

Le rodeur utilise la Sagesse pour ses sorts.

Au niveau 1 :

- sorts prepares de niveau 1+ : 2 ;
- emplacements de sort de niveau 1 : 2.

Sorts de niveau 1 recommandes dans les sources :

- frappe piegeuse ;
- soins.

- les emplacements de sort sont recuperes apres un repos long ;

### Bottes d'arme

Le rodeur choisit deux types d'arme dont il maitrise les bottes.

Exemples visibles :

- arc long ;
- epee courte ;
- cimeterre ;
- epee longue.

Apres un repos long, le rodeur peut changer les types d'arme choisis.

### Ennemi jure

Le sort `marque du chasseur` est toujours prepare.

Effets visibles :

- le rodeur peut le lancer sans depenser d'emplacement ;
- au niveau 1, il dispose de 2 utilisations gratuites ;
- les utilisations sont recuperees apres un repos long ;
- le nombre d'utilisations augmente avec les niveaux de rodeur.

## Niveau 2

### Fin explorateur

Le rodeur gagne :

- Expertise dans une competence deja maitrisee qui ne beneficie pas encore de l'Expertise ;
- deux langues au choix.

### Style de combat

Le rodeur recoit un don de style de combat au choix.

Option visible :

| Option | Effet pratique |
|---|---|
| Combattant druidique | Apprendre deux sorts mineurs de druide, consideres comme sorts de rodeur. |

Sorts mineurs recommandes pour Combattant druidique :

- assistance ;
- poussiere d'etoile.

## Niveau 3 - Sous-classe

Sous-classes mentionnees dans les sources :

- Chasseur ;

- Vagabond feerique.

Point de couverture : les pages detaillees de ces sous-classes ne sont pas visibles dans les photos 15-17. Il faudra les ajouter si un joueur choisit une sous-classe precise.

## A noter sur la fiche

- Dexterite : prioritaire pour attaque a distance, CA, initiative, discretion.
- Sagesse : prioritaire pour sorts, perception, survie et certaines aptitudes.
- Constitution : utile pour PV et concentration.
- `marque du chasseur` : toujours prepare, suivre les utilisations gratuites.
- Bottes d'arme : noter les deux armes choisies et les revoir apres repos long.
- Style de combat : a choisir au niveau 2.
