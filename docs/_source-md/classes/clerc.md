# Clerc

Sources images : 9-11.

Statut : premiere extraction faite. Sous-classes a completer si besoin.

## Role de la classe

Le clerc est un lanceur de sorts divins. Il peut soutenir, soigner, proteger, affaiblir ou chatier selon ses sorts et son domaine.

Pour une table debutante, le clerc sert souvent a :

- maintenir le groupe en vie ;
- preparer des sorts de soutien ;
- utiliser une armure et un bouclier pour rester solide ;
- gerer les morts-vivants ;
- apporter une forte Sagesse au groupe.

## Niveau 1

Au niveau 1, le clerc recoit :

- les traits de base ;
- l'aptitude Sorts ;
- l'aptitude Ordre divin.

### Sorts

Le clerc utilise la Sagesse pour ses sorts.

Au niveau 1 :

- sorts mineurs connus : 3 ;
- sorts prepares de niveau 1+ : 4 ;
- emplacements de sort de niveau 1 : 2.

Sorts mineurs recommandes dans les sources :

- assistance ;
- flamme sacree ;
- thaumaturgie.

Sorts de niveau 1 recommandes dans les sources :

- benediction ;
- bouclier de la foi ;
- rayon tractant ;
- soins.

Notes pratiques :

- les sorts prepares peuvent etre changes apres un repos long ;
- les emplacements de sort sont recuperes apres un repos long ;
- un sort prepare doit etre d'un niveau pour lequel le clerc possede un emplacement ;
- certains sorts obtenus par d'autres aptitudes peuvent ne pas compter dans la limite de sorts prepares.

### Ordre divin

Le clerc choisit une orientation sacree au niveau 1 :

| Option | Effet pratique |
|---|---|
| Protecteur | Ajoute armes de guerre et armures lourdes. |
| Thaumaturge | Ajoute un sort mineur de clerc et renforce certains tests d'Intelligence lies au sacre. |

Choix rapide :

- prendre Protecteur si le personnage doit tenir la ligne de front ;
- prendre Thaumaturge si le personnage veut etre plus tourne vers la magie et le savoir religieux.

## Niveau 2 - Conduit divin

Le clerc gagne Conduit divin au niveau 2.

Utilisations :

- 2 utilisations au niveau 2 ;
- recuperation partielle apres repos court ;
- recuperation complete apres repos long ;
- le nombre d'utilisations augmente a plus haut niveau.

Effets visibles :

| Effet | Resume |
|---|---|
| Etincelle divine | Soigner une creature ou infliger des degats radiants/necotiques dans un rayon de 9 m. Jet de base : 1d8 + modificateur de Sagesse. Ajouter 1d8 aux niveaux 7, 13 et 18. |
| Renvoi des morts-vivants | Morts-vivants choisis dans un rayon de 9 m : JS de Sagesse. En cas d'echec, etats Effraye et Neutralise pendant 1 minute, avec obligation de s'eloigner. |

Si un effet de Conduit divin demande un jet de sauvegarde, utiliser le DD de sauvegarde des sorts du clerc.

## A noter sur la fiche

- Sagesse : caracteristique prioritaire.
- Constitution : importante pour les PV et la concentration.
- CA : dependra de l'armure, du bouclier et du choix Ordre divin.
- Symbole sacre : utile comme focaliseur d'incantation.
- Sorts prepares : a revoir apres chaque repos long selon la journee prevue.
