## Role de la classe

Le druide est un lanceur de sorts lie aux forces de la nature. Il peut soigner, controler le terrain, parler aux animaux, adopter des formes animales et canaliser des effets elementaires.

Pour une table debutante, le druide sert souvent a :

- apporter de la magie de soutien et de controle ;
- fournir des soins ;
- aider en exploration et survie ;
- utiliser Forme sauvage pour s'adapter a certaines situations ;
- couvrir les competences liees a la nature, aux animaux et a la perception.

## Niveau 1

Au niveau 1, le druide recoit :

- les traits de base ;
- l'aptitude Sorts ;
- l'aptitude Druidique ;
- l'aptitude Ordre primitif.

### Sorts

Le druide utilise la Sagesse pour ses sorts.

Au niveau 1 :

- sorts mineurs connus : 2 ;
- sorts prepares de niveau 1+ : 4 ;
- emplacements de sort de niveau 1 : 2.

Sorts mineurs recommandes dans les sources, a proposer mais a ne pas considerer comme choisis tant que le joueur ne valide pas :

- druidisme ;
- flammes.

Important (couverture) : la page du descriptif de classe renvoie a une « liste des sorts de Druide, plus loin dans le descriptif » (image 12), mais cette liste **n'est pas dans les images photographiees** et le chapitre 7 (sorts complets) non plus. La liste exhaustive des cantrips de druide n'est donc pas disponible dans le projet.

Cantrips de druide attestes ailleurs dans les sources (cross-reference) :

- `assistance` et `poussiere d'etoile` sont documentes comme « sorts mineurs de druide » via la botte Combattant druidique du rodeur (`rodeur.md`, images 15-17).

Sorts de niveau 1 recommandes dans les sources, a proposer pour les 4 sorts prepares mais a ne pas considerer comme choisis tant que le joueur ne valide pas :

- amitie avec les animaux ;
- lueurs feeriques ;
- soins ;
- vague tonnante.

Notes pratiques :

- les sorts prepares peuvent etre changes apres un repos long ;
- les emplacements de sort sont recuperes apres un repos long ;
- certains sorts obtenus par d'autres aptitudes peuvent ne pas compter dans la limite de sorts prepares.

### Druidique

Le druide connait le druidique, langue secrete des druides.

Effets pratiques visibles :

- `communication avec les animaux` est toujours prepare ;
- le druidique permet de laisser des messages caches ;
- les autres personnes connaissant le druidique reperent automatiquement ces messages ;
- d'autres creatures peuvent detecter la presence d'un message avec un test d'Investigation, sans forcement le comprendre.

### Ordre primitif

Le druide choisit une orientation au niveau 1 :

| Option | Effet pratique |
|---|---|
| Gardien | Ajoute armes de guerre et armures intermediaires. |
| Mage | Ajoute un sort mineur de druide et renforce certains tests d'Intelligence lies aux arcanes et a la nature. |

Choix rapide :

- prendre Gardien si le druide doit etre plus expose en combat ;
- prendre Mage si le druide veut renforcer son role de lanceur de sorts et de connaisseur de la nature.

## Niveau 2

### Compagnon sauvage

Le druide peut utiliser la magie de la nature pour appeler un esprit animal sous forme de familier.

Effets pratiques visibles :

- demande une action Magie ;
- depense soit un emplacement de sort, soit une utilisation de Forme sauvage ;
- lance une version d'appel de familier sans composantes materielles ;
- le familier est une Fee ;
- le familier disparait apres un repos long.

### Forme sauvage

Le druide peut adopter une forme animale.

Points visibles :

- transformation par action Bonus ;
- la forme dure un nombre d'heures egal a la moitie du niveau de druide ;
- peut etre quittee par action Bonus ;
- s'arrete aussi si le druide reutilise Forme sauvage, devient Neutralise ou meurt ;
- au niveau 2, le druide dispose de 2 utilisations ;
- une utilisation est regagnee apres repos court ;
- toutes les utilisations sont regagnees apres repos long ;
- le nombre d'utilisations augmente plus tard.

Formes connues visibles :

- le druide connait quatre formes de Bete ;
- le facteur de puissance maximal indique est 1/4 ;
- pas de vitesse de vol au depart ;
- formes recommandees visibles : araignee, cheval de selle, loup, rat.

## A noter sur la fiche

- Sagesse : caracteristique prioritaire.
- Constitution : utile pour les PV et la concentration.
- Dexterite : utile pour la CA et certaines armes.
- Materiel d'herboriste : maitrise a noter.
- Focaliseur druidique : necessaire pour l'incantation.
- Forme sauvage : noter le nombre d'utilisations, les formes connues et les limites de mouvement.
