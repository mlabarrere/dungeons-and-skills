# Notes D&D

Ce dossier contient des fiches de reference personnelles construites a partir des photos du livre.

Objectif : aider a creer et faire evoluer les personnages de la table sans recopier le livre complet.

## Fichiers

- `protocole-creation-personnage.md` : methode pour creer une fiche, calculer les valeurs, detecter les trous et proposer des recommandations.
- `modele-graphe-provenance.md` : modele de graphe dirige pour tracer les relations entre sources, choix, effets et cases de fiche.
- `creation-personnage.md` : procedure de creation et decisions a prendre.
- `classes/clerc.md` : fiche pratique du clerc.
- `classes/druide.md` : fiche pratique du druide.
- `classes/rodeur.md` : fiche pratique du rodeur.
- `origines.md` : historiques, especes et choix d'origine.
- `dons.md` : index pratique des dons utiles.
- `equipement.md` : equipement de depart, armes, armures, outils, paquetages.
- `sorts-selectionnes.md` : sorts effectivement choisis par les joueurs.
- `generateur-online.md` : cadrage du futur generateur web imprimable.
