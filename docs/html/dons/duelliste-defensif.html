Benefices

  • Augmentation de caracteristique. Dexterite +1 (max 20).
  • Parade. Si vous tenez une arme avec la propriete Finesse et qu'une autre creature vous frappe avec une attaque de corps a corps, vous pouvez jouer votre Reaction pour ajouter votre bonus de maitrise a votre classe d'armure, ce qui peut faire echouer l'attaque. Vous recevez ce bonus a votre CA contre les attaques de corps a corps jusqu'au debut de votre tour suivant.
