Benefices

  • Points de chance. Vous disposez d'un nombre de points de chance egal a votre bonus de maitrise, recuperes en terminant un Repos long.
  • Avantage. Quand vous jouez un d20 dans le cadre d'un Test d20, vous pouvez depenser 1 point de chance pour vous octroyer l'Avantage a ce jet.
  • Desavantage. Quand une creature joue un d20 pour un jet d'attaque contre vous, vous pouvez depenser 1 point de chance pour lui imposer le Desavantage.
