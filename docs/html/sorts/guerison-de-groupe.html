Guerison de groupe

Abjuration du 9e niveau (Clerc)

Niveau 9 Abjuration Clerc

Incantation

Temps d'incantationAction
Portee18 m
ComposantesV, S
DureeInstantanee

Description

Vous produisez une vague d'energie curative vers les creatures qui vous entourent. Vous restituez un maximum de 700 points de vie, repartis entre les creatures de votre choix parmi celles que vous voyez a portee. Les creatures soignees par ce sort se debarrassent en outre des etats Assourdi, Aveugle et Empoisonne.
