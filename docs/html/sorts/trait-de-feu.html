Trait de feu

Evocation mineure (Ensorceleur, Magicien)

Sort mineur Evocation Ensorceleur Magicien

Incantation

Temps d'incantationAction
Portee36 m
ComposantesV, S
DureeInstantanee

Description

Vous projetez un grain enflamme vers une creature ou un objet a portee. Effectuez une attaque de sort a distance contre la cible. Si l'attaque touche, la cible subit 1d10 degats de feu. Tout objet inflammable touche par ce sort s'embrase s'il n'est pas porte.

Amelioration de sort mineur

Les degats augmentent de 1d10 lorsque vous atteignez les niveaux 5 (2d10), 11 (3d10) et 17 (4d10).
