Communication a distance

Divination du 3e niveau (Barde, Clerc, Magicien)

Niveau 3 Divination Barde Clerc Magicien

Incantation

Temps d'incantationAction
PorteeIllimitee
ComposantesV, S, M (du fil de cuivre)
DureeInstantanee

Description

Vous envoyez un bref message n'excedant pas 25 mots a une creature que vous avez rencontree (ou qu'une personne qui l'a rencontree vous a decrite). L'esprit de la cible percoit le message, celle-ci vous identifie comme expediteur et si elle vous connait et peut vous repondre aussitot dans le meme mode. Le sort permet aux cibles de comprendre le sens de votre message.

La distance qui vous separe du destinataire n'a pas d'importance et vous pouvez meme atteindre d'autres plans d'existence, mais si la cible se trouve sur un plan different, il y a 5 % de chance pour que le message n'aboutisse pas. Si la transmission ne s'est pas faite, vous le savez.

A la reception du message, la creature peut vous empecher de la recontacter par ce sort pour les 8 prochaines heures. Si vous tentez de lui envoyer un autre message dans cet intervalle, vous savez qu'elle vous a bloque et le sort echoue.
