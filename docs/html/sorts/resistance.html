Incantation

DureeConcentration, jusqu'a 1 minute

Description

Vous touchez physiquement une creature consentante en choisissant un type de degats entre : acide, contondants, feu, foudre, froid, necrotiques, perforants, poison, radiants, tonnerre et tranchants. Quand la creature subit des degats du type retenu avant la fin du sort, elle reduit les degats subis de 1d4. Une meme creature ne peut beneficier qu'une fois par tour de ce sort.
