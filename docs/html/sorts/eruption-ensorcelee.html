Eruption ensorcelee

Evocation mineure (Ensorceleur)

Sort mineur Evocation Ensorceleur

Incantation

Temps d'incantationAction
Portee36 m
ComposantesV, S
DureeInstantanee

Description

Vous projetez des energies ensorcelees vers une creature ou un objet a portee. Effectuez une attaque de sort a distance contre la cible. Si l'attaque touche, la cible subit 1d8 degats du type choisi : acide, feu, foudre, froid, poison, psychiques ou tonnerre.

Si vous obtenez un 8 sur un d8 du sort, vous pouvez en lancer un autre pour l'ajouter aux degats. Quand vous lancez ce sort, le maximum de d8 supplementaires que vous pouvez ainsi ajouter aux degats du sort est egal a votre modificateur de caracteristique d'incantation.
