Flamme sacree

Evocation mineure (Clerc)

Sort mineur Evocation Clerc

Incantation

Temps d'incantationAction
Portee18 m

Description

Une radiance embrasee s'abat sur une creature que vous voyez a portee. La cible doit reussir un jet de sauvegarde de Dexterite sous peine de subir 1d8 degats radiants. Les Abris partiels et superieurs ne conferent aucun benefice dans le cadre de ce jet de sauvegarde.
