Premonition

Divination du 9e niveau (Barde, Druide, Magicien, Occultiste)

Niveau 9 Divination Barde Druide Magicien Occultiste

Incantation

Temps d'incantation1 minute
PorteeContact
ComposantesV, S, M (une plume de colibri)
Duree8 heures

Description

Vous touchez physiquement une creature consentante pour lui octroyer la faculte d'entrevoir l'avenir immediat. Pour toute la duree, la cible a l'Avantage aux Tests d20 et les autres creatures subissent le Desavantage aux jets d'attaque contre elle. Le sort prend fin prematurement si vous le lancez a nouveau.
