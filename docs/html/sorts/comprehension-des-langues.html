Incantation

Temps d'incantationAction ou rituel
PorteePersonnelle

Description

Pour toute la duree, vous comprenez le sens litteral de toute langue parlee que vous entendez. Vous comprenez egalement tout ecrit que vous voyez, a condition de toucher la surface sur laquelle il est redige. Il vous faut environ 1 minute pour lire une page de texte. Ce sort ne decrypte pas les symboles ni les messages secrets.
