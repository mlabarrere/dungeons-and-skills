Stabilisation

Necromancie mineure (Clerc, Druide)

Sort mineur Necromancie Clerc Druide

Incantation

Temps d'incantationAction
Portee4,50 m
ComposantesV, S
DureeInstantanee

Description

Choisissez a portee une creature dotee de 0 point de vie, mais qui n'est pas morte. La creature est Stabilisee.

Amelioration de sort mineur

La portee augmente lorsque vous atteignez les niveaux 5 (9 m), 11 (18 m) et 17 (36 m).
