Description

La foudre jaillit de vous vers une creature que vous tentez de toucher. Effectuez une attaque de sort au corps a corps contre la cible. Si l'attaque touche, la cible subit 1d8 degats de foudre et ne peut pas effectuer d'attaque d'Opportunite jusqu'au debut de son tour suivant.
